# 📊 Retail Sales Performance & Discount Impact Analysis

# 🔍 Project Overview
This project presents an interactive Power BI dashboard designed to analyze retail sales performance and evaluate the impact of discount strategies on profitability. The objective is to transform raw transactional data into meaningful insights that support data-driven decision-making.

# 🎯 Objectives
* Analyze overall sales and profit performance
* Study the impact of discounts on profitability
* Identify top-performing customers and products
* Understand sales trends across time, category, and region

# 🛠️ Tools & Technologies Used
* Microsoft Power BI
* Power Query (Data Cleaning)
* DAX (Data Analysis Expressions)
* CSV Dataset (Global Superstore)

# 📂 Dataset
The dataset used is the Global Superstore Dataset obtained from Kaggle.
It includes fields such as:
* Sales
* Profit
* Discount
* Category
* Customer Details
* Region
* Order Date

# 📈 Dashboard Pages
1. Overview Dashboard
* KPI Cards (Sales, Profit, Orders, Margin, Discount)
* Sales Trend (Line Chart)
* Sales by Category and Region
* Interactive slicers

2. Discount Analysis
* Scatter Plot (Discount vs Profit)
* Column Chart (Sales by Discount Category)
* Line Chart (Profit Trend)
* Slicers for Category and Region

3. Customer & Product Insights
* Top Customers Table
* Top Products Bar Chart
* Profit Distribution (Donut Chart)
* Segment Analysis
  

# 🔑 Key Insights

* Sales increase with higher discounts, but profit decreases
* Moderate discounts provide a balance between sales and profit
* Revenue is concentrated among top customers and products
* Technology category contributes the highest profit
* Consumer segment dominates sales
